## Installation

There is no need to install anything.

[Visit the website here.](https://willfelder.github.io/learning-plan-web/)

If you prefer:

- Clone repository

```
git clone https://github.com/willfelder/learning-plan-web.git
``` 
- Use your favorite IDE
- Execute npm install
- Execute npm start
